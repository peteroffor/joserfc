Migrating from PyJWT¶

Compare the code from PyJWT:

PyJWT¶
import jwt
encoded_jwt = jwt.encode({"some": "payload"}, "secret", algorithm="HS256")

And joserfc:

joserfc¶
from joserfc import jwt
encoded_jwt = jwt.encode({"alg": "HS256"}, {"some": "payload"}, "secret")
